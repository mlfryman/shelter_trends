# shelter_trends
Determine what factors affect how long an animal stays in the shelter before adoption


## 1. Create Virtual Environment
```
cd shelter_trends
python3 -m venv venv
source venv/bin/activate
pip install pandas matplotlib seaborn jupyter streamlit scikit-learn
```

## 2. Create Notebook
```
jupyter notebook
```
Browser will load Jupyter. View existing or create new notebook in `shelter_trends/notebooks/`.
